GUIDA PER NPM:
Per installare npm, bisogna installare prima NodeJS:
link spiegazione -> https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
link node js -> https://nodejs.org/it/

1) installo nodeJS
2) per installare npm globalmente -> npm install npm -g 
3) per verificare le versioni di nodeJs e npm:
    node -v
    npm -v
4) per aggiornare npm -> npm install npm@latest -g
5) per runnare Angular 2, bisgona avere le seguenti versioni:
   Per node v4.x.x or higher e npm 3.x.x or higher 
   by running the commands node -v and npm -v in a terminal/console window.
